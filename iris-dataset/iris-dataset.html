Iris Data Set¶

Introduction¶

The Iris data set problem is a supervised classification task wherein three species of Iris (Iris setosa, Iris virginica and Iris versicolor) must be distinguished using data from four continuous features: sepal width, sepal height, petal width, and petal height. Based on the combination of these four features, Ronald Fisher developed a linear discriminant model to distinguish the species from each other.

Linear Discriminant Analysis¶

Linear discriminant analysis (LDA) involves finding a linear combination of continuous features that maximise the seperation between known categories. The result is a projection from the original feature space to a linear subspace consisting of the directions which maximize the separation between classes. A similar classifier is quadratic discriminant analysis (QDA) which learns quadratic boundaries and is better able to model more complex distributions of data. LDA also appears to be similar to Principal Component Analysis (PCA) which reduces the dimensionality of the feature space based on the distribution of data; however, PCA accounts for the variation of all data whereas LDA accounts for the variation between categories and hence is better suited for classifying categorical data.

LDA works via two rules:

  1. maximise the distance between the means of each category; and
  2. minimise the variation/scatter within each category.

To achieve this, LDA makes the following assumptions:

  1. that your data is Gaussian, that each variable is is shaped like a bell curve when plotted; and
  2. that each attribute has the same variance, that values of each variable vary around the mean by the same amount on average.

Applying LDA to the Iris Data Set Problem¶

For the Iris data set, a linear discriminant model would reduce the four feature dimensions representing sepal/petal width and height such that the distance between the means of each species is maximised and the variation of each species is minimised.

Model Creation and Evaluation¶

Next we will split the data into a training and testing set.

In [10]:
# Load train_test_split method
from sklearn.model_selection import train_test_split

# Split into training and testing sets
X_train, X_test, Y_train, Y_test = train_test_split(iris.data, 
                                                    iris.target, 
                                                    test_size=0.20)

As mentioned earlier, Fisher was able to construct a linear discriminant model to distinguish the species from one another.

We will train and test the following models:

  1. Linear Discriminant Analysis
  2. Quadratic Discriminant Analysis
In [11]:
# Load ML models
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis as LDA
from sklearn.discriminant_analysis import QuadraticDiscriminantAnalysis as QDA

# Initialise the models
models = {
    'LDA Model': LDA(),
    'QDA Model': QDA()
}

# Train the models

    models[name].fit(X_train, Y_train)

When repeated, the results vary slightly depending on the random state of the 'train_test_split' method which divides the samples between the training and testing set. Despite this, we consistently observe an accuracy rating above 96% for both the LDA and QDA model. Taking a closer look at the classification report, we find that the '1' and '2' class (representing the Versicolor and Virginica species) have a lower precision and recall in comparison to class '0' (representing Setosa), which may be because the characteristics of Versicolor and Virginica are more similar and hence more difficult to dissimilate.

Conclusion¶

The results of this analysis confirm Fisher's idea that a linear discriminant model is sufficient for the categorisation of the three species of Iris to a high degree of accuracy. The Versicolor and Virginica class are more likely to be misclassified as each other due to their similarity in characteristics, whereas the Setosa class is more dinstinct and hence less likely to be mistaken for another species.
